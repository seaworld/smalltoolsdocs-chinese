========
例子
========

最好的学习方式是例子，scrapy也不例外，基于这个原因，这有一个scrapy项目的例子叫ditbot，你可以使用和学习Scrapy，他包含dmoz爬虫的教程

dirbot_ project: https://github.com/scrapy/dirbot

它包含README，详细介绍项目内容。

如果你经常使用git，你可以检索出代码，要不你可以使用tarball或者zip下载项目。

Snipplr上的scrapy用于分享snippets一样的爬虫中间件、扩展或脚本，享受自由（鼓励）分享任何代码。
